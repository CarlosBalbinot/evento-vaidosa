# assets/

Coloque aqui os dois arquivos de logo usados no rodapé do site:

- `logo-vaidosa.png` — logo da Vaidosa Fitness
- `logo-talya.png` — logo da Talya Portaluppi (Pilates & Bem-Estar)

Recomendações para os arquivos:

- Fundo transparente (PNG ou SVG).
- Altura mínima de 200px, para ficar nítido em qualquer tela — o site
  reduz a exibição para ~44px (mobile) / ~56px (desktop) automaticamente.

Enquanto os arquivos não existirem, o `index.html` mostra o texto do
wordmark no lugar da imagem (fallback automático via `onerror`).
